# 同梱スタンプ画像

このフォルダに画像ファイル（`.png` / `.gif` / `.jpg` / `.jpeg` / `.webp` / `.svg`）を置くと、
コントロールパネルに「カスタム画像」ボタンとして並び、ワンタップで画面に流せる。

- **ファイル名がそのまま表示名・キー**になる（例: `iine.png` → `iine`）。日本語ファイル名でも可。
- 置いたあとに `yarn dev`（HMR）/ `yarn build:mac`（再ビルド）すると反映される。
- ビルド時にアプリへ同梱されるので、配布した全員が同じ画像を使える。
  - リアルタイム共有では「キー（ファイル名）」だけが配信され、各自の同梱画像で表示される。
  - そのため、共有する相手とは**同じビルド（同じ画像セット）**を使うこと。
- アニメ GIF はそのまま動く（縮小はせず原寸を高さに合わせて表示）。大きすぎる画像は表示が重くなるので、絵文字サイズ（〜128px 程度）を推奨。

> このフォルダを Git に残すためのプレースホルダーとして README を置いている。画像を追加したら一緒にコミットする。
